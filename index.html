Projetos

LigaFarma

Desenvolvimento de uma aplicação web de busca de emprego voltada para a indústria farmacêutica, similar ao LinkedIn e Glassdoor. Funcionalidades incluem: Criação de contas para farmacêuticos e farmácias, postagem e busca de vagas de emprego e recursos para candidatura e contratação. Criado utilizando Angular e Node.

Pizzaria AC

Aplicação web fullstack que simula uma pizzaria. Um frontend Angular que se comunica com uma API no backend em C# e integração com banco de dados utilizando o Entity Framework. Permite o registro e login de usuários, construção dinâmicas de pizzas, gerenciamento de um carrinho e simulação de pedidos.

Squaresmasher

Aplicação web fullstack que simula uma loja de filmes. Um frontend Angular que se comunica com uma API em um backend Spring possibilita o usuário procurar filmes para adicionar ao seu estoque e ver compras dos seus itens em estoque serem geradas.
